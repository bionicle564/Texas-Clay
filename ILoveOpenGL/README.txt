~~ GT Graphics Project 3: Additional Graphics Stuff ~~

HEY LISTEN! HEY LISTEN!
Now featuring Texture Swapping and Gradual Skybox Blending 

builds on x64 debug or x64 release

video link (youtube): https://youtu.be/aAYhwPBwcwo

Controls

W-> camera foward
S-> camera backward
A-> camera left
D-> camera right
Q-> camera down
E-> camera up
ctrl+A-> camera rotate left
ctrl+D-> camera rotate right

R -> exec next animation batch if ready

DEBUG-y Controls
K-> change debug target
I-> toggle between Light/Model debug

num8 -> debug target forwards
num5 -> debug target backwards
num4 -> debug target left
num6 -> debug target right
num7 -> debug target down
num9 -> debug target up

num+ -> Save debug target to json
num- -> Overwrite Debug target from json
num* -> Save all models to json
num/ -> read all models from json !!NOTE: DANGEROUS and probably disabled!!

ctrl+num8 -> Pitch debug target forwards
ctrl+num5 -> Pitch debug target backwards
ctrl+num4 -> Tilt debug target left
ctrl+num6 -> Tilt debug target right
ctrl+num7 -> rotate debug target down
ctrl+num9 -> rotate debug target up

num1 -> increase light constant
num2 -> increase linear atten
num3 -> increase Quadratic atten
fullstop -> increase Cutoff

ctrl+num1 -> decrease light constant
ctrl+num2 -> decrease linear atten
ctrl+num3 -> decrease Quadratic atten
ctrl+fullstop -> decrease Cutoff
